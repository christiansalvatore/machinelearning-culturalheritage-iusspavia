## Syllabus
* Machine learning and the «data life cycle»
    * Feature extraction
    * Feature selection
    * Classification

* Validation and classification metrics
   (or how to stop worrying and setup the model)

* Deep-learning techniques

* Applications to the cultural-heritage
   domain
